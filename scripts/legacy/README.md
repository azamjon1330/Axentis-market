# Legacy-скрипты

Разовые скрипты деплоя/диагностики/починки, накопившиеся в корне репозитория
за время ручного администрирования VPS. Перенесены сюда, чтобы корень проекта
оставался чистым. Они по-прежнему работают, но актуальный процесс деплоя
описан в [`docs/DEPLOY_MANUAL.md`](../../docs/DEPLOY_MANUAL.md).

Если скрипт больше не нужен — смело удаляйте.
