```powershell

# 2. Измените $env:PGPASSWORD на свой пароль
# 3. Запустите:
.\apply-missing-migrations.ps1
```

### Шаг 3: Перезапустите бэкенд
```powershell
# Остановите текущий процесс бэкенда (Ctrl+C)

cd backend
.\azaton-backend.exe

# Или если нужна пересборка:
go build -o azaton-backend.exe main.go
.\azaton-backend.exe
```

### Шаг 4: Пересоберите фронтенд (опционально)
```powershell
npm run build
# или для dev-режима:
npm run dev
```

## 🧪 Как проверить, что всё работает:

### Тест 1: Сжатие изображений
1. Откройте страницу добавления товара
2. Загрузите изображение размером > 5 МБ

### Тест 2: Копирование ключа
1. Откройте управление компаниями

## 🐛 Если проблемы остались:

### Для 500 ошибок:
1. Проверьте логи бэкенда - там будут детали ошибки
2. Убедитесь, что миграции применены: `.\check-database.ps1`
3. Проверьте консоль браузера (F12 → Network)
