## 🐛 РЕШЕНИЕ ПРОБЛЕМ:

### Проблема: 500 ошибка при отправке сообщений
**Решение:**
```powershell
# 1. Проверьте таблицы
.\check-database.ps1

# 2. Если таблиц нет, примените миграции
.\apply-missing-migrations.ps1

# 3. Перезапустите бэкенд
cd backend
.\azaton-backend.exe

# 4. Посмотрите логи бэкенда - теперь они детальные!
```

### Проблема: Ключ не копируется
**Решение:**
1. Убедитесь, что используете современный браузер (Chrome, Firefox, Edge)
2. Проверьте, что сайт открыт по HTTPS (для Clipboard API)
3. Если не работает, появится alert с самим ключом - скопируйте вручную
4. Альтернатива: выделите текст мышью и Ctrl+C

### Проблема: Изображение не загружается
**Решение:**
1. Откройте консоль (F12)
2. Посмотрите ошибку в логах
3. Проверьте путь к файлу
4. Убедитесь, что бэкенд запущен
5. Проверьте, что файл существует в папке `backend/uploads/`
