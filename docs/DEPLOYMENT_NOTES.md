```powershell
.\deploy-to-vps.ps1
```

### Вариант 2: Вручную
1. Подключитесь к VPS:
   ```bash
   ssh root@109.123.253.238
   # Пароль: Supreme001
   ```

2. Выполните команды:
   ```bash
   cd /root/Axentis-market
   git pull origin main
   docker-compose build frontend --no-cache
   docker-compose up -d --force-recreate frontend
   docker ps
   ```

### Проверка
После деплоя зайдите на:
- **Аналитика:** https://axentis.uz/#/analytics
  - Проверьте что карта "Sarflangan" синего цвета
  - Должен быть виден селектор периода с заголовком
  - Графики имеют описания

- **Импорт товаров:** Перейдите в "Raqamli ombor" → нажмите кнопку импорта
  - Выберите файл Excel
  - Проверьте что все надписи переведены на узбекский
  - Попробуйте переключить язык на русский - должно работать в обоих языках
